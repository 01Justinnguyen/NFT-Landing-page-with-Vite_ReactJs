# NFT LadingPage with Vite_ReactJs by Liamdev

## Requirements

- **nodejs >= v14.7.0**
- **npm >= v6.14.7**
- **yarn >= v1.19.1**

You must install Vite and Sass in your local before clone project and run test.

```
$ git clone https://git_url_clone <project_dir>
$ cd vite-project
$ npm run dev
```
